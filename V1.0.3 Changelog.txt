SoapsQuest V1.0.3 Changelog
============================

Release date: May 2026

Overview
--------
This update focuses on premium progression systems, cleaner config design, and better feature consistency between manual quests and generated quests.

New headline feature: Sigils
----------------------------
1. Added a brand new custom progression currency called Sigils.
2. Added admin commands to manage Sigils:
   /sq sigils give <player> <amount>
   /sq sigils take <player> <amount>
   /sq sigils set <player> <amount>
   /sq sigils reset <player>
   /sq sigils balance <player>
3. Added PlaceholderAPI support:
   %soapsquest_player_sigils%
   %soapsquest_player_sigils_raw%
4. Added external API access through SoapsQuestAPI for plugin developers.

Rewards and conditions rework
-----------------------------
1. Manual quests and generated quests now share the same reward structure.
2. Added native Sigils rewards with reward.sigils and optional sigils-chance.
3. Added Sigil quest requirements:
   min-sigils
   sigil-cost
4. Reward handling is now reused across more systems, including recurring completion bonuses.

Random generator improvements
-----------------------------
1. Added temporary generated quest mode:
   random-generator.temporary-quests
2. Temporary generated quests are removed from the system after the player claims rewards.
3. Expanded random generation objective support with missing objective types added to config and weights.
4. Added canonical reward templates in random-generator.yml so reward blocks can be copied between manual and generated setups directly.

Daily and weekly recurring update
---------------------------------
1. Simplified recurring configuration with selection.random and selection.count.
2. Added daily and weekly completion bonus rewards using the same reward format as quests.
3. Added tracking for cycle completion and one-time bonus payout per cycle.

Quest drop command
------------------
1. Added premium drop command to spawn quest papers in the world:
   /sq drop <questId>
   /sq drop <questId> <x> <y> <z> [world]
   /sq drop entity <entity|uuid> <questId>
2. Supports both direct coordinate drops and entity-location drops.

Premium and free split
----------------------
1. New premium features are gated behind premium checks.
2. Free jar remains clean and locked from premium systems as expected.

Config, examples, and docs
--------------------------
1. Updated quests.yml with Sigils examples and requirement usage.
2. Updated daily.yml with simplified selection config and bonus reward examples.
3. Updated random-generator.yml with temporary quest mode and canonical reward template examples.
4. Updated command and placeholder documentation for Sigils and new premium commands.
5. Added TEST-NEW-FEATURES.md for quick smoke testing of all new functionality.

Build output
------------
1. Premium release built:
   SoapsQuest-1.0.3-Premium.jar
2. Free release built:
   SoapsQuest-1.0.3-Free.jar
