SHORT DESCRIPTION

Hold a quest in your hand - a physical paper item with 37 objective types, quest chains, GUI browser, tiers, conditions, and MiniMessage support.


WHAT IS SOAPSQUEST?

SoapsQuest turns quests into real items in your players' inventories. Instead of menus and invisible trackers, every quest is a physical paper item players can hold, hover over for progress, and right-click to claim rewards when they're done.

There's no NPC setup, no complex dependencies, and no database to configure. Drop the jar in your plugins folder, restart, and you have a working quest system with example quests ready to go.

Quests are defined in simple YAML. You write what players need to do, what rewards they get, and optionally what conditions they need to meet. The plugin handles everything else - progress tracking, notifications, quest papers, and reward delivery.


CORE CONTENT

Everything included in the free version:

- 37 Objective Types - Kill, mine, place, craft, smelt, fish, tame, brew, enchant, trade, breed, harvest, shear, explore biomes, gain levels, collect XP, travel, elytra fly, ride vehicles, shoot arrows, launch projectiles, repair anvils, interact with blocks, and more
- Physical Quest Papers - Quests are real inventory items. Hover for progress, right-click to claim rewards
- Quest Browser GUI - In-game menu to browse and pick up available quests
- Active Quests GUI - View all current active quests in a dedicated GUI
- Flexible Rewards - XP, items (custom names, lore, enchantments), money (Vault), commands, or quest chains
- Condition System - Lock quests behind level, world, permission, money cost, item cost, gamemode, or active quest limits
- Tiers and Difficulties - Customizable tier categories and difficulty levels with reward multipliers
- Sequential Objectives - Force objectives to complete in order
- Milestones - Notify players at configurable progress percentages
- MiniMessage Formatting - Gradients, hex colors, bold, italic, hover/click events in all text
- Full Admin Tools - Give, remove, reset, complete, copy, reload, statistics, and debug commands
- PlaceholderAPI, Vault, and MythicMobs Support - Optional integrations for placeholders, economy, and custom mobs
- Customizable Messages and Sounds - Every message and sound is configurable

INSTALLATION

1. Download SoapsQuest.jar
2. Place it in your server's plugins/ folder
3. Restart your server (a full restart, not /reload)
4. Edit quests in plugins/SoapsQuest/quests.yml
5. Run /sq reload to apply changes
6. Give a quest to yourself: /sq give <yourname> lumberjack

The plugin ships with 10 ready-to-use example quests so you can see how everything works immediately.


FILES

After first run, these files are generated in plugins/SoapsQuest/:

config.yml         - Core plugin settings (progress display, autosave, sounds, quest limits)
quests.yml         - All quest definitions
messages.yml       - Every plugin message, fully customizable
tiers.yml          - Quest tier names, colors, and item icons
difficulties.yml   - Difficulty levels and reward multipliers
gui.yml            - Quest browser and active quests GUI layout and appearance


COMMANDS

/sq help - Show available commands
/sq info - Show plugin version and details
/sq give <player> <questid> - Give a quest paper to a player
/sq list - List all available quest IDs
/sq browse - Open the quest browser GUI
/sq active - Open your active quests GUI
/sq abandon <questid> - Abandon an active quest
/sq reload - Reload all config files
/sq addreward <quest> <type> <value> - Add a reward to a quest
/sq removereward <quest> <type> - Remove a reward from a quest
/sq listreward <quest> - List rewards on a quest
/sq remove <player> <questid> - Remove a quest from a player
/sq reset <player> <questid> - Reset a player's quest progress
/sq complete <player> <questid> - Force-complete a quest for a player
/sq copy <from> <to> <questid> - Copy a quest from one player to another
/sq statistic [player] - View quest completion statistics
/sq debug - Show debug information

SUPPORTED PLUGINS

SoapsQuest works standalone with no required dependencies. These optional plugins add extra features:

Vault              - Money rewards and money-cost conditions
