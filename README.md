# SoapsQuest

SoapsQuest is a quest plugin for Paper/Spigot servers running Minecraft 1.21 and above.

Players receive physical quest papers as items in their inventory. They carry the paper while completing tasks and right-click it to claim rewards when done. No complicated menus or commands needed for players.

[Discord](https://discord.gg/mawAzwFq) | [Website](https://SoapsUniverse.com) | [GitHub](https://github.com/AlternativeSoap/SoapsQuest)

---

## Premium Features

Some features in SoapsQuest require the **Premium version**. These are clearly marked throughout the documentation with a note like this:

> **[PREMIUM]** This feature requires the SoapsQuest Premium version. Get it at [SoapsUniverse.com](https://SoapsUniverse.com)

**Premium features include:**
- Random Quest Generator
- Daily and Weekly Quests
- Quest Loot System (mob drops and chest loot)
- In-Game Quest Editor GUI

Everything else is available in the free version.

## Requirements

- Paper or Spigot 1.21 or newer
- Java 21 or newer

## Optional Add-ons

These are not required, but unlock extra features if installed:

| Plugin | What It Adds |
|--------|--------------|
| Vault | Money rewards and the money cost condition |
| PlaceholderAPI | SoapsQuest data in scoreboards, holograms, and other plugins |
| MythicMobs | The "kill mythic mob" objective type |
