SHORT DESCRIPTION

Hold a quest in your hand - everything in the free version plus Random Quest Generator, Daily/Weekly Quests, Quest Loot System, and an In-Game Quest Editor.


WHAT IS SOAPSQUEST PREMIUM?

SoapsQuest Premium is the complete version of SoapsQuest, a physical quest paper plugin for Paper 1.21+. It includes every feature from the free version - 37 objective types, quest chains, conditions, tiers, difficulties, GUI browser, MiniMessage formatting, and full admin tools - plus four major premium systems:

Random Quest Generator - Generate quests automatically from configurable templates. Define objective pools, name patterns, reward ranges, and rarity weights. Creates unique quests on demand or on a schedule.

Daily and Weekly Quests - Automatically distribute rotating quests on a schedule. Configure reset times, quest pools, quests per cycle, and notification style. Offline players receive theirs on next login.

Quest Loot System - Quest papers drop from mobs and appear in loot chests. Configure per-mob drop chances, quest pools, and level requirements. Works with vanilla loot tables.

In-Game Quest Editor GUI - Create and edit quests from an in-game GUI. No YAML editing required. Add objectives, rewards, and conditions by clicking through menus. Saves directly to config.

Premium: Random Quest Generator

- Generate unique quests from weighted templates on demand or automatically
- Define objective pools with types, targets, and amount ranges
- Configure name templates with randomized elements
- Set reward ranges for XP, money, and items
- Multi-objective generated quests supported
- Rarity weighting for quest type frequency
- MythicMobs support in generated kill objectives
- Milestone auto-generation for longer quests

Premium: Daily and Weekly Quests

- Auto-distribute quest papers on a daily or weekly schedule
- Configure exact reset times (24-hour format)
- Define quest pools - the plugin picks from the list each cycle
- Randomize selection or cycle in order
- Configure how many quests per reset
- Offline players receive quests on next login
- No duplicates until the full pool has cycled
- Notification modes: title screen, action bar, chat, or any combination

Premium: Quest Loot System

- Quest papers drop from mobs as loot
- Per-mob drop chances and quest-to-mob mappings
- Quest papers appear in dungeon and structure loot chests
- Works with vanilla loot tables and specific chest types
- Configurable min/max player level requirements for drops
- Weight-based quest selection from loot pools

Premium: In-Game Quest Editor GUI

- Create new quests from a point-and-click GUI
- Edit existing quests without touching YAML
- Add, remove, and modify objectives, rewards, and conditions
- Set name, tier, difficulty, material, lore, and all options
- Saves directly to quests.yml
- Requires soapsquest.gui.editor permission


INSTALLATION

1. Download SoapsQuest-Premium.jar
2. Place it in your server's plugins/ folder
3. Restart your server (a full restart, not /reload)
4. Edit quests in plugins/SoapsQuest/quests.yml or use /sq editor in-game
5. Run /sq reload to apply config changes
6. Give a quest to yourself: /sq give <yourname> lumberjack

The plugin ships with 10 ready-to-use example quests so you can see how everything works immediately.


FILES

After first run, these files are generated in plugins/SoapsQuest/:

config.yml           - Core plugin settings (progress display, autosave, sounds, quest limits)
quests.yml           - All quest definitions
messages.yml         - Every plugin message, fully customizable
tiers.yml            - Quest tier names, colors, and item icons
difficulties.yml     - Difficulty levels and reward multipliers
gui.yml              - Quest browser, active quests, and editor GUI layout and appearance
daily.yml            - Daily and weekly quest schedules and settings
quest-loot.yml       - Mob drop and chest loot quest paper settings
random-generator.yml - Random quest generation templates and pools


COMMANDS

All free commands plus premium additions:

/sq help - Show available commands
/sq info - Show plugin version and details
/sq give <player> <questid> - Give a quest paper to a player
/sq list - List all available quest IDs
/sq browse - Open the quest browser GUI
/sq active - Open your active quests GUI
/sq abandon <questid> - Abandon an active quest
/sq reload - Reload all config files
/sq addreward <quest> <type> <value> - Add a reward to a quest
/sq removereward <quest> <type> - Remove a reward from a quest
/sq listreward <quest> - List rewards on a quest
/sq remove <player> <questid> - Remove a quest from a player
/sq reset <player> <questid> - Reset a player's quest progress
/sq complete <player> <questid> - Force-complete a quest for a player
/sq copy <from> <to> <questid> - Copy a quest from one player to another
/sq statistic [player] - View quest completion statistics
/sq debug - Show debug information
/sq generate <player> - Generate a random quest for a player (Premium)
/sq editor [questid] - Open the quest editor GUI (Premium)
