# Changelog

## 1.0.1 — May 2026

### Fixed
- **command** and **placeholder** objectives now track progress correctly
- **explore_biome** works on Paper 1.21+ (registry key matching)
- Direct objectives respect **active queue**, **conditions**, and **sequential** order
- Queued papers no longer gain progress while inactive
- Fish/craft-style objectives no longer double-count progress

### Improved
- Command rewards: `{player}`, `<player>`, `%player%`
- `completed-quests` condition as a flat YAML list
- 37 `showcase_*` test quests in default `quests.yml`
- `/sq reload` reports quest count; `/sq info` shows Free vs Premium

### Download
- **Free** and **Premium** from [SoapsUniverse.com](https://soapsuniverse.com) and [MythicCraft.io](https://mythiccraft.io)

## 1.0.0 — Initial release

First public release with physical quest papers, 37 objectives, tiers, conditions, GUI browser, and Premium features (generator, daily/weekly, loot, editor).
