# Changelog

## 1.0.1

### Fixed
- `command` and `placeholder` objectives track progress correctly
- `explore_biome` works on Paper 1.21+ biome IDs
- Direct objectives respect active queue, conditions, and sequential order
- Queued papers no longer gain progress while waiting
- Fish/craft-style objectives no longer double-count

### Improved
- Command rewards accept `{player}`, `<player>`, `%player%`
- `completed-quests` condition as a simple YAML list
- 37 `showcase_*` test quests in default `quests.yml`
- `/sq reload` shows quest count; `/sq info` shows Free or Premium

https://soapsuniverse.com and https://mythiccraft.io
