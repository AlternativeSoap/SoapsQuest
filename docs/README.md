# SoapsQuest documentation

Quest plugin for Paper 1.21+. Version **1.0.1**.

Download from https://soapsuniverse.com or https://mythiccraft.io. You need **SoapsCommon** on the server as well.

New here? Start with [Getting Started](Getting-Started.md), then [Creating Quests](Creating-Quests.md) and [Objectives](Objectives.md).
